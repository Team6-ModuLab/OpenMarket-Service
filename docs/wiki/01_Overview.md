# 01. Overview - 프로젝트 개요

## 프로젝트 목적

**호두마켓(HODU)**은 오픈마켓 형태의 이커머스 웹 서비스입니다. 구매자와 판매자가 상품을 거래할 수 있는 플랫폼을 순수 HTML/CSS/JavaScript로 구현한 프론트엔드 프로젝트입니다.

## 주요 기능

### 구매자 기능
- 상품 목록 조회 및 검색
- 상품 상세 정보 확인
- 장바구니 관리 (추가, 수량 조절, 삭제)
- 상품 주문

### 판매자 기능
- 판매자 센터에서 상품 관리
- 상품 등록/수정/삭제

### 공통 기능
- 회원가입 (구매자/판매자)
- 로그인/로그아웃
- 배너 슬라이더 (상품 목록 페이지)

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일링 | CSS3 (CSS Variables, Flexbox) |
| 스크립팅 | Vanilla JavaScript (ES6+) |
| API 통신 | Fetch API |
| 상태 관리 | localStorage |

## 전체 페이지 목록

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 상품 목록 | `/pages/products/list/` | 메인 페이지, 배너 슬라이더 + 상품 그리드 |
| 상품 상세 | `/pages/products/detail/` | 상품 정보, 장바구니 추가, 바로 구매 |
| 장바구니 | `/pages/cart/` | 장바구니 아이템 관리, 주문하기 |
| 주문 | `/pages/order/` | 주문 정보 확인 |
| 로그인 | `/pages/auth/login/` | 사용자 인증 |
| 회원가입 | `/pages/auth/signup/` | 새 계정 생성 (구매자/판매자) |
| 판매자 센터 | `/pages/seller/seller-center/` | 판매자 상품 관리 |
| 상품 등록 | `/pages/seller/seller-product-upload/` | 상품 등록/수정 |

## 프로젝트 구조 요약

```
OpenMarket-Service/
├── index.html              # 진입점 (리다이렉트)
├── pages/                  # 페이지별 컴포넌트
│   ├── auth/              # 인증 (로그인, 회원가입)
│   ├── products/          # 상품 (목록, 상세)
│   ├── cart/              # 장바구니
│   ├── order/             # 주문
│   └── seller/            # 판매자 기능
└── shared/                 # 공용 리소스
    ├── css/               # 공용 스타일
    ├── js/                # 공용 스크립트
    └── assets/            # 아이콘, 이미지
```

## 사용자 유형

| 유형 | userType | 접근 가능 기능 |
|------|----------|---------------|
| 구매자 | `BUYER` | 상품 조회, 장바구니, 주문 |
| 판매자 | `SELLER` | 상품 조회, 판매자 센터, 상품 등록/수정/삭제 |
| 비로그인 | - | 상품 조회만 가능 (장바구니 접근 시 로그인 모달) |
